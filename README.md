# HelloDev Events

ScriptableObject-based event system for decoupled communication. Supports typed events (void, bool, int, float, string) with auto-reset between play sessions.

## Installation

Add the package to Unity Package Manager -> Add package from Git URL:

$installUrl

(For a specific release, add #v1.0.0 to the URL after tagging a release.)

## Usage

Import into your project and follow the HelloDev docs.
